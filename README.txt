Winged Lion — مشروع جاهز للرفع على GitHub Pages
----------------------------------------------

محتويات الحزمة:
- index.html  : صفحة HTML تعرض الأسد المجنح على خلفية شفافة، بها تحريك للرأس والأجنحة ودوران بسيط.
                 تحتوي أيضًا على مُركب صوتي (WebAudio) يقوم بتوليد زئير بشكل مُصطنع.
- README.txt  : تعليمات سريعة للرفع على GitHub Pages.

كيفية التشغيل محليًا (لتجربة سريعة):
1. فك ضغط المحتوى.
2. افتح index.html مباشرة في المتصفح (Chrome/Firefox). 
   - ملاحظة: بعض المتصفحات تمنع تشغيل الصوت التلقائي دون تفاعل المستخدم. 
     إن حدث ذلك، اضغط زر 'تشغيل الصوت' في واجهة الصفحة.

رفع على GitHub Pages (لينك دائم):
1. سجل دخولك في GitHub. اذهب إلى صفحة: https://github.com/new
2. أنشئ Repository جديد باسم: winged-lion
   - اختر Public.
3. ارفع الملفات الموجودة في المجلد (يمكنك سحب وإفلات الملفات مباشرة).
4. بعد الرفع، اذهب للإعدادات (Settings) > Pages > Source > اختر main branch و folder: / (root) ثم Save.
5. بعد ثوانٍ ستحصل على رابط مشابه: https://mohfathi2014-prog.github.io/winged-lion/
   افتحه وسيعمل المشهد شفافًا (خلفية شفافة) وسيحاول تشغيل الزئير تلقائيًا.

ملاحظات تقنية:
- التصميم الحالي مولد هندسيًا ضمن Three.js (لا يوجد ملف GLB خارجي). 
- لو حبيت نسخة أكثر واقعية (نموذج GLB عالي التفاصيل مع خرائط PBR)، أقدر أجهزها ونرفعها لاحقًا.
- تشغيل الصوت التلقائي قد يُمنع في متصفحات معينة — زر 'تشغيل الصوت' موجود كبديل.
