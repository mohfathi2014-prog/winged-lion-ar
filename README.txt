Winged Lion — حزمة جاهزة للرفع على GitHub Pages

محتويات الحزمة:
- index.html  : صفحة HTML تعرض الأسد المجنح على خلفية شفافة، مع حركة رأس، رفرفة أجنحة، ودوران.
                 تحتوي أيضًا على تركيب صوتي مُصطنع (WebAudio) يحاول التشغيل التلقائي.
- README.txt  : تعليمات سريعة للتحميل والرفع على GitHub Pages.

كيفية التشغيل محليًا (تجربة سريعة):
1. فك ضغط المحتوى.
2. افتح index.html في متصفح حديث (Chrome أو Edge أو Firefox).
   - ملاحظة: المتصفحات تمنع أحيانًا التشغيل التلقائي للصوت؛ استخدم زر 'تشغيل الصوت' داخل الصفحة إذا لم يبدأ الصوت تلقائيًا.
   - لا حاجة لوجود ملفات اضافية — المشهد مُنشأ برمجياً داخل HTML.

رفع على GitHub Pages (لينك دائم):
1. افتح المستودع الخاص بك في GitHub (مثال: https://github.com/mohfathi2014-prog/winged-lion-ar).
2. اضغط 'Add file' → 'Upload files' ثم اسحب كل ملفات الحزمة (index.html و README.txt) وضعهم في المربع.
3. اضغط 'Commit changes' لرفع الملفات إلى الفرع main.
4. فعل GitHub Pages من Settings → Pages → اختر Branch: main و Folder: / (root) ثم Save.
5. افتح الرابط العام: https://mohfathi2014-prog.github.io/winged-lion-ar/

ملاحظات تقنية:
- التصميم الحالي مولد هندسياً بواسطة Three.js داخل الصفحة (لا يحتوي على ملف GLB خارجي)، وهذا يجعل الحزمة أخف وأسهل للرفع.
- لو رغبت بنسخة GLB أكثر تفصيلاً لاحقًا أقدر أجهزها.
